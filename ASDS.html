ASDS — make your config line

Fill these in, click Generate, then copy the line and paste it into your GitHub file (instructions below).

How to use it:
1. On GitHub, open your ASDS.html file → click the pencil (Edit).
2. Press Ctrl/Cmd+F in the editor, find: id="bakedConfig"
3. You'll see >null< between the script tags. Replace the word null with the whole line you copied.
4. Commit changes. Wait 1 minute. Done — staff link is now configured.
